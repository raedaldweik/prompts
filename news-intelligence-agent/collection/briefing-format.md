# Intelligence Briefing Format (Sample)

## Default structure
**Headline takeaway** — 1–2 sentences: the single most important thing.

**Key developments** — 3–6 bullets. Each bullet:
> *What happened* — **Source**, *date* — *why it matters to RTA.*

**Watch / next** — 1–3 short bullets on emerging items to track.

**Sources** — a clean list of the links used.

## Worked example (illustrative)
**Headline takeaway:** The UAE tightened distracted-driving penalties this week,
signalling stricter enforcement that fleets should prepare for.

**Key developments:**
- New higher fines for handheld phone use announced — *The National, 2026-06-26* — fleet policies and driver coaching may need updating.
- Regional robotaxi pilot expanded to a new district — *Reuters, 2026-06-24* — relevant to RTA's AV roadmap and public-acceptance work.

**Watch / next:**
- Draft V2X standards expected next quarter.

**Sources:**
- https://example-source-one
- https://example-source-two

## Rules
- Keep each bullet to one or two lines; lead with the fact, end with the RTA angle.
- Always include source + date; never an undated claim.
- Order by importance, then recency.
- If little was found, say so honestly and narrow or widen the search next time.
