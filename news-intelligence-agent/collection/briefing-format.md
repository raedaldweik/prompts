# Intelligence Briefing Format (Sample)

## Default structure
**Headline takeaway** — 1–2 sentences: the single most important thing.

**Key developments** — 3–6 bullets. Each bullet:
> *What happened* — **Source**, *date* — *why it matters to NCGR.*

**Watch / next** — 1–3 short bullets on emerging items to track.

**Sources** — a clean list of the links used.

## Worked example (illustrative)
**Headline takeaway:** New digital-contracting features announced for Etimad
this week signal faster award-to-contract cycles that entities should prepare
their workflows for.

**Key developments:**
- Digital contracting workshop held with the Federation of Saudi Chambers — *Etimad portal, 2026-06-26* — supplier onboarding materials may need updating.
- OECD published its updated bid-rigging detection guidance — *OECD, 2026-06-24* — worth folding the new red-flag list into the integrity methodology.

**Watch / next:**
- Expected Ministry of Finance circular on local-content scoring next quarter.

**Sources:**
- https://example-source-one
- https://example-source-two

## Rules
- Keep each bullet to one or two lines; lead with the fact, end with the NCGR angle.
- Always include source + date; never an undated claim.
- Order by importance, then recency.
- If little was found, say so honestly and narrow or widen the search next time.
